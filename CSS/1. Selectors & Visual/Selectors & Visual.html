Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. inline styling; je voegt de styling code direct toe bij het html element. body style="background-color= red;" >

  3. in separate document styling; je kan de css in een apart .css bestand en in de header hier naar linken met een link element.

Opdracht 3

Er zijn verschillende selectors, geef in je antwoord ook aan wat de voordelen zijn van elke selector:

- type selectors: Dit is een type selector om alle html selectors te selecteren en te kunnen stylen. Snelle manier om (grote) groepen elementen te kunnen stylen.

- descendant selectors: Dit type selector kan handig zijn om alle elementen binnen een bepaald element te kunnen stylen. Je kunt zo grotere groepen selecteren en toch wat specifieker zijn.

- class selectors: Hiermee kun je heel precies bepaalde elementen selecteren en stylen.

Opdracht 8

Leg in eigen woorden uit wat met cascade en inheritance in CSS wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

- Met de cascade in de style sheets wordt bedoelt dat de browser de code regel voor regel, van boven naar onder leest en dat de onderste regel bij een dubbele toewijzing geldt. Verder is hier de 'specificity' van belang. Bij conflicterende toewijzing van code is de meest specifieke geldend, de relevantste die het dichtbij staat en zo een soort score krijgt toegewezen die uiteindelijk bepaalt wat aan wie wordt toegewezen.
- Met de inheritance (wat overerving betekend) word bedoelt dat indien een element wordt gestyled dat sommige styling worden meegenomen, waaronder de background-color en font-styling. Maar bijvoorbeeld 'width' wordt niet meegenomen.
