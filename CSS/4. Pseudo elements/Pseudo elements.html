::part()

Deze pseudo-element selector gaat

custom-element::part(foo) { /* Styles to apply to the `foo` part */ }

::placeholder

Dit wordt voornamelijk gebruikt bij in put velden. Dit om het veld een kleur te geven indien een input is gevalideerd en dus ook ingevoerd. Daarnaast kan je het ook gebruiken als extra (hint) text in het input veld.


::selection

Als je deze tekst selecteerd dan zie je wat het pseudo ::selection kan doen met de geselecteeerde tekst.


::slotted()

Dit gaat in combinatie met een HTML slot element en javascript.
