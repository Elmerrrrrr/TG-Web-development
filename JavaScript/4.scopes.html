Opdracht 4

Wat wordt er in de console gelogt? Hoe is dit anders voorgaande opdrachten? 
        function printCar() {
            carBrand = "Volvo";
            
            return function(type) {
                console.log(carBrand + " " + type);
            }
        }
  
        const car = printCar(); // De waarde van deze variabele is wat er gereturned word uit de functie en is in dit geval een functie.
        console.log(car);
        car("v40"); //de functie wordt gestart en geeft een argument mee.
De waarde van de variabele 'car' is een geneste functie met een parameter 'type'. Deze word vervolgens gestart met een argument 'v40' en geconsole logt samen met de variabele 'carBrand' die in de functie er buiten is gedeclareerd maar wel toegangkelijk is. (Lexical Scope)
