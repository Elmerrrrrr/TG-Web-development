Opdracht 7

Hoe geef je 2 of meer parameters mee aan een functie? Geef een voorbeeld. Wat zijn rest parameters? Geef een voorbeeld. Wat is het verschil tussen rest parameters en expliciete parameters die je aangeeft bij een functie?
Je kan een functie meerdere waardes (argumenten) in de parameters van de functie meegeven (ongelimieerd en gescheiden met een ,). Deze kunnen in de functie als variabele gebruikt worden, en daarna eventueel met een nieuwe waarden worden gereturned. Maar als we meer argumenten meegeven dan er parameters zijn worden alleen de gedefineerde parameters meegenomen in de functie. Om er voor te zorgen dat indien er meer argumenten(waardes) worden meegegeven aan de functie dan dat er parameters zijn gedefinieerd kan er gebruikt gemaakt worden van een rest parameter die als laaste parameter in de functie wordt gedefineerd doormiddel van drie puntjes: 
    // function with rest (which makes an array of the rest items)
    function assignment7Part2(a,b,...c){ 
        console.log(`${a} ${b}`); //Mukul Latiyan 
        console.log(c);  //[ 'Lionel', 'Messi', 'Barcelona' ] 
        console.log(c[0]); //Lionel 
        console.log(c.length); //3 
        console.log(c.indexOf('Lionel')); //0 
    } 
    assignment7Part2('Mukul','Latiyan','Lionel','Messi','Barcelona'); 

    // same function with no rest
    function assignment7Part1(a,b,c){ 
        console.log(`${a} ${b}`); //Mukul Latiyan 
        console.log(c);  // Lionel  
        console.log(c[0]); //L 
        console.log(c.length); //6 but this is the word Lionel now!
        console.log(c.indexOf('Lionel')); //0 it is not an array anymore.
    } 
    assignment7Part1('Mukul','Latiyan','Lionel','Messi','Barcelona');
