Opdracht 9

Wat is de uitkomst? ++
Het cijfer 3 wordt verhoogt met 1 (++) dus: 4
Wat is de uitkomst? --
Het cijfer 3 wordt verlaagt met 1 (--) dus: 2
Wat is de uitkomst? cijfer++ +2 (post-increment) // evaluates the value, then increments and stores it.
Bij post-increment wordt de waarde gereturned voor dat er 1 bij op wordt geteld. Het antwoord is dus 5
Wat is de uitkomst? ++cijfer +2 (pre-increment) // increments the value, then evaluates and stores
Bij pre-increment wordt het cijfer 3 wordt verhoogt met 1 en vervolgens gereturned en er wordt daarna nog 2 bij opgeteld dus: 6

Maak een eigen voorbeeld van de volgende functies: charAt(), concat()

                //charAt()
                let str = "abcd efgh" ;
                let character = str.charAt(5);
                console.log(character); // output is e, space is not counted.
        
                //concat()
                const str1 = "abc";
                const str2 = "def";
                console.log(str1.concat(str2));
                console.log(str1.concat('-', str2)); //je kan doormiddel van een komma nog meer samenvoegen
                
                //slice()
                const str3 = "Lotje leerde lopen langs de lange Lindenlaan";
                console.log(str3.slice(6));// spaties worden hierin wel meegeteld.
                console.log(str3.slice(6, 13));// "leerde" is de output hier
                console.log(str3.slice(-10));// "Lindenlaan" is de output hier
                console.log(str3.slice(-19,-10 ));// "de lange" is de output hier
                
                //split()
                const str4 = "Lotje leerde lopen langs de lange Lindenlaan";
                const words = str4.split(' '); // the space in the brackets selects all words and puts them in an array.
                console.log(words[3]); // output is the 4th word, starts at 0.
        
                const letter = str4.split(''); // without the space in the brackets selects all words and puts them in an array.
                console.log(letter[2]); // output is the 3th letter, starts at 0.
        
                const strCopy = str4.split(); // nothing inside the brackets defines all text
                console.log(strCopy); // output is the complete string
        
                //substr()
                const str5 = 'Mozilla';
                
                console.log('substr2:' + str5.substr(2)); //output "zilla", removes an number of caracther from the string.
                console.log('substr:' + str5.substr(1, 5)); // output: "ozill", last index number is included
