Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.
De CSS position property heeft een aantal type values, namelijk:
  1. Static: (default value van position) hier heeft het element de "normale" positie
  2. Relative: Hier kun je nog extra waardes meegeven met top, right, bottom en left, relatief aan de orginele positie. Dit veranderd niet de positie van elementen hier om heen.
  3. Absolute: Hier kun je het element relatief positioneren tenopzichte van het parent element doormiddel van top, right, bottom en left.
  4. Fixed: Hiermee positioneer je het element ten opzichte van de viewport, ofwel de browser window.
  5. Sticky: Hiermee positioneer je een element ten opzichte van zijn dicht bij zijnde parent (relative) met top, right, bottom en left. Dan blijft het plakken bij zijn dicht bij zijnde parent in een fixed positie. Zie het navigatie menu in op deze pagina.
